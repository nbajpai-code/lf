# 🏗️ Your Personal Learning Operating System

> *"Knowledge without a system is just noise. Build the system once, and it pays dividends forever."*

---

## What Is a Learning OS?

A Personal Learning Operating System (PLOS) is a **structured, repeatable system** combining the best techniques into a daily and weekly practice. Instead of randomly applying techniques when you remember them, a PLOS makes evidence-based learning your default mode.

This document synthesizes all techniques in this repository into a unified, actionable framework.

| Pillar | Technique | Frequency |
|---|---|---|

## 🗓️ The Daily Learning Rhythm

### Morning Protocol
```
□ Consistent wake time (circadian anchor)
□ Exercise (30 min aerobic — BDNF + cortisol clearance)
□ Review SRS deck (15–20 min Anki/RemNote — before new content)
□ Set ONE learning goal for the day (Pareto filtered)
```

### Deep Learning Blocks (Peak Hours)
```
Block 1 (90 min): New material — hardest topic first
  - 10 min: Review yesterday's notes (active recall, no re-reading)
  - 60 min: New content (reading, video, course)
  - 20 min: Feynman teach-back (write or explain aloud)

[15 min break — walk, hydrate, no screens]

Block 2 (60 min): Practice and application
  - 45 min: Deliberate practice problems / exercises
  - 15 min: AI sparring — quiz, challenge, analogies

[Lunch — avoid heavy carbs]
```

### Afternoon Optional Block
```
60 min: Interleaved review
  - 20 min: Earlier topic (spaced retrieval)
  - 20 min: Current topic practice
  - 20 min: Earlier topic again (mixed)
```

### Evening Protocol
```
□ Brain dump (10 min): Write everything learned today, no notes
□ Flashcard creation (5–10 min): 3–5 cards from today's key concepts
□ Light review of tomorrow's topic (10 min — pre-exposure)
□ No screens 30 min before sleep
□ 7–9 hours sleep (consolidation engine runs overnight)
```

## 🗺️ The Topic Onboarding Process

When starting any new topic, run this **Metalearning Sprint** first (1–2 days):

```
Phase 0: Metalearning (before studying content)
  1. AI Prompt: "Map the territory of [TOPIC]" — get overview
  2. Identify Pareto 20%: What are the core concepts I must master?
  3. Choose resources: 1 book + 1 course + 1 community
  4. Identify a concrete project to build/complete
  5. Set a timeline: when will I test my mastery?
```

Only then begin the daily learning rhythm.

## 🛠️ Minimum Viable PLOS

If you can only implement 3 things, implement these:

```
1. FILTER: Before studying, identify the Pareto 20% with an AI metalearning prompt
2. CARD: After every session, make 3–5 flashcards (use Anki — free)
3. REVIEW: 15 minutes every morning reviewing SRS before new content
```

This alone will 2–3× your learning retention vs. typical passive study.
