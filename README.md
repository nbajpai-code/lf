## 🧠 Philosophy

> *"The goal is not to know more facts — it is to build better mental models."*

Most people learn slowly because they:
- **Re-read** instead of **recall**
- **Highlight** instead of **connect**
- **Study everything** instead of **filtering the vital 20%**
- **Learn in isolation** instead of **teaching and applying**

This repository is your operating manual to learn 2–5× faster using scientifically proven strategies — amplified by AI.

## 🚀 Quick Start: The Power Stack

If you only have time for three techniques, combine these:

```
1. Pareto Filter     → Identify the 20% of content that gives 80% of value
2. Feynman Pass      → Explain it in plain language to expose gaps
3. Spaced Repetition → Review at optimal intervals to lock it in forever
```

Add **AI-powered prompting** to get personalized quizzes, analogies, and flashcards on demand.

## 🗺️ Learning Speed Formula

```
Learning Speed = (Focus Quality × Retrieval Frequency × Sleep Quality)
                 ────────────────────────────────────────────────────
                 (Cognitive Load × Distraction × Poor Spacing)
```

**Maximize the numerator. Reduce the denominator.**
